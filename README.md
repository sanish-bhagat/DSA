# DSA Course in C++

Welcome to my **Data Structures and Algorithms (DSA) Course** repository in **C++**.  
This repo is a complete collection of my learning journey and code implementations for mastering DSA in C++.  
It includes well-structured folders covering each major topic.

## 🚀 Topics Covered

- Arrays and Strings
- Linked Lists (Singly, Doubly, Circular)
- Stacks and Queues
- Trees (Binary Trees, BST, Heap)
- Graph
- Sorting and Searching algorithms
- Recursion and Backtracking
- Hashing and HashMaps
- Important coding interview problems

## 💻 Language

All code examples and practice problems are implemented in **C++**.
